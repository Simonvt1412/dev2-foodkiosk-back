API Documentatie

API Overzicht

De FoodKiosk API biedt toegang tot de volgende endpoints:

  • GET /api/categories - Alle categorieën ophalen
  • GET /api/products/:categoryId - Producten per categorie ophalen
  • POST /api/orders - Nieuwe bestelling plaatsen
